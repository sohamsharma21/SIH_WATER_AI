# SIH WATER AI - Documentation

This directory contains all project documentation.

## Files

- **README.md** (main project README) - Project overview and setup instructions
- **ARCHITECTURE.md** - System architecture and design details
- **API_DOCS.md** - Complete API endpoint documentation
- **SIH_presentation.pptx** - Auto-generated SIH presentation (run `scripts/generate_sih_presentation.py` to regenerate)

## Generating Documentation

### SIH Presentation

To generate the SIH presentation:

```bash
cd scripts
python generate_sih_presentation.py
```

The presentation will be saved to `docs/SIH_presentation.pptx`.

Note: You may want to add screenshots and custom images to the presentation manually.
